Learning

So what did I learn with these article/section boxes?

  1. Check and check again for typos in your class/id names. So many times I either changed a name in the html and forgot in the css, or I made a typing error on one page.
  2. Sometimes you do need to nest flex-containers. I have a .flex-container for the main central 3-column layout (blue background which is visible only when I have errors). The middle column (red) is also a flex container.
  3. Styling text elements like heading and paragraph tage and images is done in the flex-container for the flex-items that contain them. This pink section is the flex-item. The red div is its container. The html elements here belong to the container.
  4. Many of my errors this week were due to my lack of understanding of this.
  5. I needed to make the red and green divs flex-basis 70% and 15%(each green)
  6. The items within the containers are fixed width relative to the container. So each pink section within the red is width: 90% of the red. (with margins of 15px and ppadding of 10px.
